### Usage:
1. copy `TWLFix-CFW.3dsx` to `SD:/3ds/`
2. open homebrew launcher
3. run TWLFix-CFW
4. follow the on screen prompts.
   - once complete, the app should reboot the 3ds.
6. go to `settings -> other settings -> system update`
